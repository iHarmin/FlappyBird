## Requirements

- Python 3.x

---

## Setup Instructions

### 1. Clone the Repository
- git clone https://github.com/iHarmin/FlappyBird.git
- cd FlappyBird

### 2. Install Dependencies
- pip3 install pygame

### 3. Run the Game
- python3 flappybird.py
